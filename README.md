# GRAMMAR PRO · 英文文法互動教學

為國高中生打造、可用於 **多益 / 學測 / 商用英文** 的互動式文法教材。單一網頁、離線可用、手機電腦全響應式。**8 大主題、44 堂課、108 題**，完整涵蓋學測與多益核心文法考點。

🔗 **線上使用（Live Demo）**：<https://job159.github.io/Grammar/>

## 特色
- **8 大主題、44 堂課**：動詞時態、語態與語氣、子句與連接、詞類與用法、比較與修飾、句型結構、動詞句型、多益商用。
- **全頁講義式閱讀**：點課程進入獨立全頁版面（雜誌級排版、閱讀進度條、右側課程地圖、上一課/下一課），非彈窗側欄。
- **每課九段式結構**：對話式開場鉤子 → 圖解（時間軸 SVG／對照表／流程圖）→ 公式結構 → 關鍵訊號字 → 例句拆解 → 考點提示 → 常見錯誤 → 師生問答（對話式教學）→ 考前 30 秒重點回顧 → 隨堂測驗。
- **五大分頁**：📖 學習 / ✍️ 測驗 / 🎯 實戰 / 📕 錯題本 / 📊 進度。
- **🔊 發音朗讀（TTS）**：課程英文主題、每則例句、測驗題目、錯題本皆可一鍵朗讀（Web Speech API，離線可用、免金鑰）。
- **📕 錯題本**：答錯的題目自動收集、答對後自動移除，可一鍵「重練錯題」，導覽列即時顯示錯題數徽章。
- **🎯 實戰模擬考**：學測英文 / 多益 Part 5 / **克漏字題組** 三種題型，**3,100+ 題大題庫**（隨機出題），可選**難度（基礎／進階／精熟）**，限時作答、答題面板可跳題、一次交卷、及格判定與**逐題詳解**（標示難度，錯題自動入錯題本）。
- **📄 克漏字題組**：多益 Part 6／學測綜合測驗題型，**252 篇 × 4 格 = 1,008 題**（12 篇手寫＋20 個手寫題框以「文法變體 × 詞彙置換」生成 240 篇），商用 email／公告／廣告＋學測記敘說明文，含**整句插入題**；146 種正解形式、每格附「規則＋誘答逐項拆解」詳解；選項即時填入文章、整篇對答案，錯的空格自動入錯題本。
- **💡 答題詳解 + 難度分級**：每題都附文法詳解與難度標籤，練習回饋、實戰檢討、錯題本皆完整顯示。
- **互動與動畫**：捲動浮現、卡片展開側欄、選項即時對錯回饋、通關彩帶、進度環、亮／暗主題切換。
- **測驗模式**：可選單一主題或綜合測驗（108 題題庫、隨機出題），即時批改與解析，記錄最佳成績。
- **進度追蹤**：以 `localStorage` 記錄已研讀課程、作答數、正確率、各主題掌握度、模擬考最佳成績與錯題本。

## 涵蓋文法考點
| 主題 | 課程 |
|------|------|
| 動詞時態 | 現在簡單/進行、現在完成、完成進行、過去進行、過去完成、未來三式、未來完成/進行、時態一致 |

| 比較與修飾 | 比較級最高級、進階比較、形容詞副詞、分詞形容詞 |
| 句型結構 | 五大句型、倒裝句、強調句、附加問句、主詞動詞一致、分詞構句 |
| 動詞句型 | 使役動詞、感官動詞、動詞片語、特殊動詞句型 |
| 多益商用 | 商用情態助動詞、多益陷阱、連接副詞、詞性變化 |

## 使用方式
直接用瀏覽器開啟 `index.html` 即可，無需安裝任何套件。

如需本機伺服器預覽：
```bash
python -m http.server 5510
# 開啟 http://localhost:5510
```

## 擴充內容
在 `data.js` 的 `LESSONS` 陣列中新增課程物件即可，欄位格式：
```js
{
  id, cat,          // cat 對應 CATEGORIES 的 id
  level,            // 基礎 / 進階 / 精熟
  title, en, summary,
  structure: [{ label, formula, tone }],   // tone: a / b / c
  examples:  [{ en, zh, note? }],
  tips:      [ "..." ],
  pitfalls:  [{ wrong, right, why }],
  quiz:      [{ q, opts:[...], a, ex }],    // a = 正確選項索引
}
```
新增後畫面與測驗題庫會自動更新。
